
# LUCCA MESSENGER

Front test to integrate LUCCA. Made with Agular.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Benoit2109/lucca-front.git
```

Go to the project directory

```bash
  cd lucca-front
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve
```
